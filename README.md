# CyberPulse AI

## Overview
CyberPulse AI is an AI-powered cybersecurity threat detection system that analyzes network traffic and predicts cyber attacks using Machine Learning.

## Features

- AI-based Network Traffic Analysis
- Random Forest Classifier
- Threat Detection
- Threat Severity Classification
- AI Recommendations
- Modern Dashboard
- Flask REST API

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python
- Flask

Machine Learning:
- Scikit-learn
- Random Forest

Dataset:
- NSL-KDD Dataset

## How to Run

Backend

cd backend
python app.py

Frontend

Open index.html using Live Server.
